### backend/routes/rateCardRoutes.js
**API endpoints for rate card**

Endpoints Provided:
- GET /api/rate-cards (list all)
- POST /api/rate-cards (create)
- GET /api/rate-cards/:cardId (get one)
- PUT /api/rate-cards/:cardId (update)
- DELETE /api/rate-cards/:cardId (archive)
- POST /api/rate-cards/:cardId/entries (bulk add rates)
- POST /api/quotes/generate (calculate quote)
- GET /api/quotes/analytics (get metrics)

Each endpoint includes:
- Authentication
- Input validation
- Error handling
- Console logging

**Use**: Copy to `backend/routes/rateCardRoutes.js` in Week 4. Add to server.js:
```javascript
const rateCardRoutes = require('./routes/rateCardRoutes');
app.use('/api/rate-cards', rateCardRoutes);
```
